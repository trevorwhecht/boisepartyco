What does "managing inventory" mean to you?

Select all the areas you need to manage from the admin dashboard. This helps us scope the feature correctly.

A

Item Catalog CRUD

Create new items, edit names / prices / descriptions / specs, toggle active/inactive, set sort order. The core product manager workflow.

B

Category Management

Add / edit / reorder categories (Tents, Chairs, Tables, Décor…). Currently categories are seeded but there's no UI to manage them.

C

Tent BOM Editor

For each tent configuration, define which tent parts it requires and how many. Currently bomComplete is false on all configs — this is what fills it in.

D

Serialized Units

Track individual physical assets by serial number (tent panels, poles, specific chairs). Set status: available / damaged / in_repair / decommissioned.

E

Image Management

Upload / reorder / delete photos for each item. Currently images are set via URLs; this would add upload support (via Cloudinary or similar).
