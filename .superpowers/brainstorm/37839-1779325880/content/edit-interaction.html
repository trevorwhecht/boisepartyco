How should qty editing work?

Each option below shows what happens when an admin wants to change a quantity.

A

Inline — click the number to edit

The qty cell is always an input. Admin types a new number and presses Enter (or clicks away). Saves instantly. Fast, spreadsheet-like feel.

Pros

  • Fastest for bulk qty updates
  • No extra clicks

Cons

  • Easy to accidentally change a value
B

Row edit — pencil icon, then Save

The row is read-only by default. Admin clicks a pencil icon to unlock that row for editing, then clicks Save to confirm. Prevents accidental changes.

C

Side sheet — click row, edit in panel

The table is always read-only. Clicking any item row opens a slide-in sheet on the right with the full item detail: qty, active toggle, price, and (later) images. Save from the sheet.
