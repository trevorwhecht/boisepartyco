How should the Inventory view be organized?

The view lives inside the admin dashboard (sidebar → "Inventory"). Pick the layout that best fits your workflow.

A

Horizontal Tabs — one per category

Tabs across the top: Chairs · Tables · Linens · Décor · Heaters · Floors · Catering · Lighting · Tents. Each tab shows a table of items in that category with inline qty editing.

B

Single Scrollable Page — category sections

All categories on one page, each collapsed by default. Click a header to expand and inline-edit that category's quantities. Good for a quick top-to-bottom audit.

C

Category Cards → Drill-in

Landing screen shows a grid of category cards (Chairs, Tables, Tents…) with a quick count of items + total units owned. Click a card to drill into that category's edit table.
